# 2D Platformer Game - Godot Project

## Overview
This project is a 2D platformer game created using the Godot Engine. The game features an interactive player, collision environments, and objects. The assets used in this project are sourced from a tutorial video by Brackeys.

## Features
- Interactive player character
- Collision detection with environments and objects
- Engaging platformer mechanics
- Visual assets for characters and environments

## Setup
1. **Download Godot Engine**: Ensure you have the latest version of the Godot Engine installed. You can download it from the [official website](https://godotengine.org/download).

2. **Clone the Repository**: Clone this repository to your local machine.

    ```sh
    git clone <repository_url>
    cd <repository_directory>
    ```

3. **Import the Project**:
    - Open Godot Engine.
    - Click on the "Import" button.
    - Navigate to the cloned project directory and select the `project.godot` file.
    - Click "Import & Edit".

4. **Run the Game**:
    - In Godot, press the "Play" button to run the game.
    - Ensure all assets are correctly loaded and the game runs without errors.
